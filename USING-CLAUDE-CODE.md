# 🚀 คู่มือใช้งานสำหรับผู้ที่ไม่ใช่โปรแกรมเมอร์ (ผ่าน Claude Code)

คู่มือนี้เขียนสำหรับคนที่ **ไม่เคยเขียนโปรแกรม** แต่อยากติดตั้งและใช้งานเว็บทำข้อสอบ ACT นี้

**หัวใจสำคัญ:** คุณไม่ต้องจำหรือพิมพ์คำสั่งใด ๆ เอง — แค่พิมพ์บอก **Claude Code** เป็นภาษาคนธรรมดา
แล้วมันจะจัดการให้ ด้านล่างนี้มี "ประโยคสำเร็จรูป" ให้คัดลอกไปวางใน Claude Code ได้เลย

---

## สิ่งที่คุณจะได้

- เว็บทำข้อสอบ ACT เสมือนจริง 4 พาร์ท (มีข้อสอบตัวอย่างพร้อมใช้ 215 ข้อ)
- ระบบตรวจให้คะแนนอัตโนมัติ + เฉลยพร้อมคำอธิบาย
- ระบบหลังบ้านสำหรับเพิ่ม/แก้ข้อสอบ และดูผลสอบของผู้ที่มาทำ

## สิ่งที่ต้องมีก่อน

1. คอมพิวเตอร์ (Mac หรือ Windows)
2. **Claude Code** ติดตั้งเรียบร้อย (คุณกำลังใช้อยู่แล้ว ✓)
3. โปรแกรม **Node.js** — ถ้ายังไม่มี ไม่ต้องกังวล ให้ Claude ช่วยเช็ก (ดูขั้นตอนล่าง)

---

## วิธีติดตั้ง (บอก Claude Code ทีละขั้น)

เปิด Claude Code ขึ้นมา แล้วพิมพ์ประโยคเหล่านี้ทีละอัน รอให้ทำเสร็จก่อนค่อยพิมพ์อันถัดไป

**ขั้นที่ 0 — เช็กความพร้อมของเครื่อง** (ทำครั้งแรกครั้งเดียว)
> ช่วยเช็กว่าเครื่องนี้มี Node.js และ git ไหม ถ้ายังไม่มีช่วยบอกวิธีติดตั้งแบบง่าย ๆ ให้หน่อย

**ขั้นที่ 1 — ดาวน์โหลดโปรเจกต์**
> ช่วย clone โปรเจกต์จาก https://github.com/wphansit/act-practice-test มาไว้ในเครื่อง แล้วเข้าไปในโฟลเดอร์นั้นให้ด้วย

**ขั้นที่ 2 — ติดตั้งและเตรียมข้อสอบ**
> ช่วยติดตั้ง dependencies, สร้างไฟล์ .env.local จาก .env.example โดยสุ่มค่า SESSION_SECRET ให้ และตั้ง ADMIN_PASSCODE เป็นรหัสที่ฉันจะใช้เข้าระบบหลังบ้าน แล้วโหลดข้อสอบตัวอย่างเข้าฐานข้อมูล
>
> (พิมพ์รหัสผ่านที่อยากใช้ต่อท้าย เช่น "...ตั้ง ADMIN_PASSCODE เป็น mypassword123")

**ขั้นที่ 3 — เปิดใช้งาน**
> ช่วยรันเว็บนี้ แล้วบอก URL ที่ฉันเปิดในเบราว์เซอร์ได้

จากนั้นเปิดเบราว์เซอร์ไปที่ **http://localhost:3000** — จะเห็นหน้าเริ่มทำข้อสอบ 🎉

> 💡 ตราบใดที่ยังอยากใช้เว็บ ให้เปิดหน้าต่าง Claude Code ทิ้งไว้ (เว็บทำงานอยู่เบื้องหลัง)
> เมื่อเลิกใช้ พิมพ์บอก Claude ว่า "ช่วยปิดเว็บให้หน่อย" ได้เลย

---

## เข้าระบบหลังบ้าน (เพิ่มข้อสอบ / ดูผลสอบ)

1. เปิด **http://localhost:3000/admin**
2. ใส่รหัสผ่านที่ตั้งไว้ (ค่า ADMIN_PASSCODE จากขั้นที่ 2)
3. เพิ่ม/แก้ข้อสอบ, ดูผลสอบของคนที่มาทำ, ส่งออกเป็นไฟล์ Excel — ทำผ่านหน้าเว็บได้เลย ไม่ต้องแตะโค้ด

## งานที่ทำบ่อย — คัดลอกประโยคไปบอก Claude Code

| อยากทำ | พิมพ์บอก Claude Code แบบนี้ |
|---|---|
| เพิ่มข้อสอบทีละหลายข้อ | "ช่วยเพิ่มข้อสอบพาร์ท Reading เรื่อง <หัวข้อ> จำนวน 10 ข้อ ให้ตรงตามรูปแบบในโฟลเดอร์ db/seed" |
| เปลี่ยนรหัสผ่าน admin | "ช่วยเปลี่ยน ADMIN_PASSCODE ในไฟล์ .env.local เป็น <รหัสใหม่>" |
| ล้างผลสอบทั้งหมด | "ช่วยล้างผลสอบทั้งหมดในฐานข้อมูล แต่เก็บข้อสอบไว้" |
| อัปเดตโค้ดล่าสุด | "ช่วยดึงโค้ดเวอร์ชันล่าสุดจาก GitHub มาให้หน่อย" |
| อยากให้คนอื่นเข้าเว็บได้ | "ฉันอยากเผยแพร่เว็บนี้ให้คนอื่นเข้าได้ ช่วยแนะนำวิธีที่ง่ายและปลอดภัยที่สุด" |

## ถ้ามีอะไรพัง — ไม่ต้องกลัว

คัดลอกข้อความสีแดง (error) ทั้งหมดที่เห็น ไปวางให้ Claude Code แล้วพิมพ์ว่า:

> เจอ error นี้ ช่วยดูให้หน่อยว่าเกิดอะไรขึ้นและช่วยแก้ให้ด้วย

Claude Code จะช่วยวินิจฉัยและแก้ให้ พูดกับมันเหมือนคุยกับช่างที่นั่งข้าง ๆ ได้เลย

---

## ⚠️ ข้อควรระวังด้านความปลอดภัย

- ไฟล์ **`.env.local`** มีรหัสผ่านของคุณ — **อย่าส่งให้ใคร อย่าอัปโหลดขึ้นอินเทอร์เน็ต** (ระบบตั้งค่าไว้แล้วไม่ให้ไฟล์นี้ขึ้น GitHub)
- **อย่าวางรหัสผ่านหรือ token ในแชทสาธารณะ** หรือที่ที่คนอื่นเห็นได้
- ข้อสอบในระบบเป็นของต้นฉบับเพื่อการศึกษา (วิทยาทาน) — ถ้าจะเพิ่มข้อสอบเอง อย่าคัดลอกข้อสอบ ACT จริงหรือของผู้อื่นที่มีลิขสิทธิ์
